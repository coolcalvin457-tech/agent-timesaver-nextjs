# AGENT: Timesaver
### promptaiagents.com

A personalized AI workflow recommendation tool. Users enter their job title, answer 2–3 questions, and receive 5 tailored AI workflows plus a weekly time savings estimate.

---

## Setup (5 steps)

### 1. Install Node.js
If you don't have Node.js installed, download it from [nodejs.org](https://nodejs.org). Get the "LTS" version.

### 2. Install dependencies
Open Terminal, navigate to this folder, and run:

```
npm install
```

### 3. Add your API key
- Find the file `.env.local.template` in this folder
- Make a copy of it and rename the copy to `.env.local`
- Open `.env.local` and replace `your_api_key_here` with your actual Anthropic API key
- Save the file

Your `.env.local` should look like this:
```
ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxxxxxx
```

**Important:** Never share this file or paste your API key anywhere public.

### 4. Run the development server
```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser. You should see your site.

> If your API key isn't set yet, the tool still works — it uses built-in example responses so you can test the full experience.

### 5. Deploy to Vercel (go live)

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and connect your GitHub account
3. Import your repository
4. In your Vercel project settings, go to **Settings > Environment Variables**
5. Add a variable: Name = `ANTHROPIC_API_KEY`, Value = your API key
6. Deploy — your site will be live at a `.vercel.app` URL
7. Connect your custom domain (`promptaiagents.com`) in Vercel's Domains settings

## How the tool works

1. **Screen 00 — Intro:** Teaches AI Lesson #1 before any input
2. **Screen 01 — Job Title + Branch:** User enters title and chooses Path A (with job description) or Path B (title only)
3. **Screen 01A — JD Upload (Path A only):** Drag-and-drop or paste job description
4. **API Call 1:** Questions are generated by Claude based on job title/JD
5. **Screens 02–04 — Questions:** 2 questions (Path A) or 3 questions (Path B)
6. **API Call 2:** Workflows and ROI are generated by Claude based on all inputs
7. **Screen 06 — Results:** 5 free workflows + ROI card + email capture
8. **Screen 07 — Gate:** Soft upgrade offer (only if email was skipped in Screen 06)
9. **Screen 08 — Confirmation:** Next steps and community links

## Swapping your photo

In `app/page.tsx`, find the section with `guide-avatar`. Replace the basketball emoji with an `<img>` tag pointing to your photo:

```tsx
<img src="/calvin.jpg" alt="Calvin" style={{ width: 200, height: 200, borderRadius: 20, objectFit: 'cover' }} />
```

Place your photo in the `/public` folder.
